Spherical Harmonics in MATLAB

Main file: Harmonics.m (Execute this code)

Function to evoluate Legendre polynomial at given point (say x) : Legendre_Poly.m
Function to evoluate Associate Legendre polynomial at given point (say x) : Asso_Leg.m
Function for coordinate transformation, from spherical to cartesian: Shperical2Cartesian.m

Note: All files should be in the same folder.
